* select 选择: 基本不用,不稳定
* bubble 冒泡: 基本不用,太慢
* insert 插入: 样本小且基本有序的时候使用
--------------------------------
* shell 希尔排序:改进的插入排序
* merge 归并排序:
        - java对象排序的默认算法(使用到递归的概念),对象排序一定要稳定
        - 现在使用的是TimSort(多路归并), 使用了归并排序和二分插入排序
